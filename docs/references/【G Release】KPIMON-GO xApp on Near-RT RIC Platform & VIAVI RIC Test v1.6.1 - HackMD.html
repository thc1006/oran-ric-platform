3. Integration

3-1 RIC Test: Run the Scenario Generator

Execution:
Go to Runtime -> Test Control
Press Run test

Check:
Go to Reports -> E2 Message Flows
Press ASN1-dump-for gNB-123456

  • E2 Setup

Note:
Please check the E2 Setup Procedure has passed successfully. (Receive the E2Setup Response Message)

If no, please debug the connection between RIC Test and E2Term or between E2Term and E2mgr.

If pass successfully, you can go to next step Execute KPIMON-GO xApp.

3-2 Execute KPIMON-GO xApp

Download KPIMON-GO source code



git clone https://github.com/o-ran-sc/ric-app-kpimon-go.git -b g-release
cd ric-app-kpimon-go

Revise KPIMON-GO xApp source code

Build image

sudo docker build -t nexus3.o-ran-sc.org:10004/o-ran-sc/ric-app-kpimon-go:1.0.1 .

Onboard KPIMON-GO xApp

dms_cli onboard --config_file_path=deploy/config.json --shcema_file_path=deploy/schema.json
